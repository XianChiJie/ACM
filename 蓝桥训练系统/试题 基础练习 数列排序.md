## 试题 基础练习 数列排序

资源限制

时间限制：1.0s  内存限制：512.0MB

问题描述

　　给定一个长度为n的数列，将这个数列按从小到大的顺序排列。1<=n<=200

输入格式

　　第一行为一个整数n。
　　第二行包含n个整数，为待排序的数，每个整数的绝对值小于10000。

输出格式

　　输出一行，按从小到大的顺序输出排序后的数列。

样例输入

5
8 3 6 4 9

样例输出

3 4 6 8 9



```c++
#include<iostream>
using namespace std;
int main()
{
	int a[205], i, j, t, n;
	cin >> n;
	for (i = 1; i <= n; i++) cin >> a[i];
	for (i = 1; i <= n; i++) {
		for (j = 1; j <= n - i; j++) {
			if (a[j] > a[j + 1]) {
				t = a[j]; a[j] = a[j + 1]; a[j + 1] = t;
			}
		}
	}
	for (i = 1; i <= n; i++) cout << a[i] << " ";
	return 0;
}
```
