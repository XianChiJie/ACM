实验1-4-输入输出格式化控制 输出三角形 (5分)

本题要求编写程序，输出指定的由“*”组成的三角图案。

### 输入格式：

本题无输入

### 输出格式：

按照下列格式输出由“*”组成的三角图案。

```
****
***
**
*
```

```c++
#include<iostream>
using namespace std;
int main()
{
	cout << "****" << endl << "***" << endl << "**" << endl << "*" << endl;
	return 0;
}
```
