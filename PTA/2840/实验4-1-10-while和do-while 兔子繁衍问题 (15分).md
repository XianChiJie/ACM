实验4-1-10-while和do-while 兔子繁衍问题 (15分)

一对兔子，从出生后第3个月起每个月都生一对兔子。小兔子长到第3个月后每个月又生一对兔子。假如兔子都不死，请问第1个月出生的一对兔子，至少需要繁衍到第几个月时兔子总数才可以达到*N*对？

### 输入格式:

输入在一行中给出一个不超过10000的正整数*N*。

### 输出格式:

在一行中输出兔子总数达到*N*最少需要的月数。

### 输入样例:

```in
30
```

### 输出样例:

```
9
```



```c++
#include<iostream>
using namespace std;
int main()
{
	int n, x = 1, y = 0, z = 0, i;
	cin >> n;
	for (i=1; z < n&&n!=1; i++) {
		z = x + y;
		y = x;
		x = z;
    }
	cout <<i << endl;
	return 0;
}
```
