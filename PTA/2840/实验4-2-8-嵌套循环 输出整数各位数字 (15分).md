实验4-2-8-嵌套循环 输出整数各位数字 (15分)

本题要求编写程序，对输入的一个整数，从高位开始逐位分割并输出它的各位数字。

### 输入格式：

输入在一行中给出一个长整型范围内的非负整数。

### 输出格式：

从高位开始逐位输出该整数的各位数字，每个数字后面有一个空格。

### 输入样例：

```in
123456
```

### 输出样例：

```out
1 2 3 4 5 6 
```



```c++
#include<iostream>
using namespace std;
int main()
{
	string x;
	cin >> x;
	int n = x.length();
	for (int i = 0; i < n; ++i) printf("%c ", x[i]);
	return 0;
}
```
