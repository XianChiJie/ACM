实验1-6 -输入输出格式化控制 输出带框文字 (5分)

本题要求编写程序，输出指定的带框文字。

### 输入格式：

本题无输入

### 输出格式：

按照下列格式输出带框文字。

```
************
  Welcome
************
```



```c++
#include<iostream>
using namespace std;
int main()
{
	cout << "************" << endl << "  Welcome" << endl << "************" << endl;
	return 0;
}
```
