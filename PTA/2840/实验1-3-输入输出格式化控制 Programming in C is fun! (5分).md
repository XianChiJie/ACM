实验1-3-输入输出格式化控制 Programming in C is fun! (5分)

本题要求编写程序，输出一个短句“Programming in C is fun!”。

### 输入格式:

本题目没有输入。

### 输出格式:

在一行中输出短句“Programming in C is fun!”。

```c++
#include<iostream>
using namespace std;
int main()
{
	cout << "Programming in C is fun!" << endl;
	return 0;
}
```
