实验2-1-7 -表达式 整数152的各位数字 (10分)

本题要求编写程序，输出整数152的个位数字、十位数字和百位数字的值。

### 输入格式：

本题无输入。

### 输出格式：

按照以下格式输出：

```
152 = 个位数字 + 十位数字*10 + 百位数字*100
```



```c++
#include<iostream>
using namespace std;
int main()
{
	cout << "152 = 2 + 5*10 + 1*100" << endl;
	return 0;
}
```
