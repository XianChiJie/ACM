实验1-5-输入输出格式化控制 输出菱形图案 (5分)

本题要求编写程序，输出指定的由“A”组成的菱形图案。

### 输入格式：

本题无输入

### 输出格式：

按照下列格式输出由“A”组成的菱形图案。

```
  A
A   A
  A
```



```c++
#include<iostream>
using namespace std;
int main()
{
	cout << "  A" << endl << "A   A" << endl << "  A" << endl;
	return 0;
}
```
